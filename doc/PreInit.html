preinit_options[R]

Hash of controls affecting how the constructor functions.

:default

Any. Default value to assign if an instance variable is specified without one (e.g. new({}, 'varname')).

:on_NameError

Symbol. Action to take if an invalid instance variable name appears in the hash.

:raise

A NameError exception will be raised identifying the invalid name.

:ignore

The key/value pair with the invalid name will be silently ignored.

:convert

An attempt will be made to make the name valid (e.g., replacing illegal characters with ‘_’, etc.).

new[(options[, *args])] { |obj,*args| block } => object

Description

Create a new instance of the current class, and set instance variables in it according to the key/value pairs in any hashes that were passed.

Arguments

*args

Array of Hash (zero or more). An optional collection of name/value pairs. The names (keys) will be treated as names of instance variables, and the values as their initial contents.

options

Hash. Control settings for how the PreInit constructor should handle its operation when processing *args. (See the preinit_options section for details.)

Examples

class Foo
  include PreInit
end
ex_1 = Foo.new(:ivar1 => 1, 'ivar2' => ['an', 'array'])
=> #<Foo:0xb7551b50 @ivar2=["an", "array"], @preinit_options={}, @ivar1=1>

ex_2 = Foo.new({ :op1 => 1 }, 17) { |o,*args|
  args.each_with_index do |arg,i|
    o.instance_variable_set("@new_ivar_#{i}".to_sym, arg)
  end
}
=> #<Foo:0xb7547de4 @preinit_options={:op1=>1}, @new_ivar_0=17>

def initialize(*args)
  @preinit_options = {}
  if (block_given?)
    if (args[0].kind_of?(Hash))
      @preinit_options = args.shift
    end
    yield(self, *args)
  else
    load_attrs(*args) unless (args.empty?)
  end
end

def load_attrs(*args)
  while (arg = args.shift)
    next unless (arg.kind_of?(Hash))
    arg.each do |attr,val|
      ivar = attr
      #
      # TODO: Vet method name
      #
      # Here is where we need to worry about a key not being a
      # valid method name (like 'foo-bar' => 'val').  What to do with
      # such?
      #
      setmeth = "#{ivar.to_s}=".to_sym
      #
      # If there's already a 'foo=' method, use it rather than
      # just setting the instance variable directly -- thus preserving
      # any special processing the class has for the variable.
      #
      if (self.respond_to?(setmeth))
        self.send(setmeth, val)
      else
        self.instance_variable_set("@#{ivar.to_s}".to_sym, val)
      end
    end
  end
end
